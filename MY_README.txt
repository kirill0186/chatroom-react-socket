Запуск программы
npm install
В разных терминалах: npm start -s
		     node server/server

Алгоритм работы программы
Открывается страница входа в чат. Имеются 2 опции: создать новый чат и зайти в существующий. Есть проверка на некорректный ввод. Сохраняется последнее введённое имя пользователя.
Далее открывается страница с чатом, где можно найти ID комнаты, по которому можно присоединоться к чату, кнопку покинуть чат, список пользователей онлайн и сам чат с сообщениями и полем ввода.
При входе и выходе пользователей появляются соответствующие сообщения. 

Комментарий в коде практически нет, т.к старался писать "говорящие" названия функциям и переменным.
